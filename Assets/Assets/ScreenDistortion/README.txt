
Instructions:
-------------
1. Open Unity and import this folder into your Assets directory.
2. Inside the 'ScreenDistortion' folder, you will find:
   - The 'DistortionEffect.cs' script
   - A prefab named 'ScreenDistortionEffect.prefab'
   - A RenderTexture asset
3. Drag the prefab into your scene.
4. To trigger the effect, call `PlayEffect()` on the DistortionEffect component.

Note:
-----
This package contains a fully working screen distortion prefab.
